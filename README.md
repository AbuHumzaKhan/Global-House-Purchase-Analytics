# Global-House-Purchase-Analytics

This project provides SQL-based analytics on a simulated dataset of global house purchases. The analysis covers key housing market insights, affordability, and customer purchase behavior across different countries, cities, and property types.

# 📂 Project Structure

Global House Purchase Analytics.sql → Contains all SQL queries for dataset analysis.

# 📊 Key Analyses Performed

Dataset Overview

Total records, distinct countries, and cities.

Customer Purchase Behavior

Purchase decision distribution (% Yes/No).

Impact of furnishing status on purchase probability.

EMI-to-income ratio and its effect on decisions.

Customer salary groups vs purchase probability.

Neighbourhood rating and connectivity score impact.

Effect of crime rate & legal issues on buying decisions.

Customer satisfaction score vs purchase rate.

Market Insights

Average house prices by country and property type.

Affordability index: Price-to-salary ratio by country.

Top cities with highest average property prices.

Property size vs price correlation.

KPI Dashboard View

Average price, income, purchase rate, neighbourhood rating, and connectivity score by country.

Top 10 cities with highest average property prices.

# 🛠️ Technologies Used

SQL Server / MySQL (queries are ANSI-compliant but optimized for SQL Server syntax).

Dataset assumed: global_house_purchase_dataset.

# 🚀 How to Use

Import/load the dataset into your SQL environment (e.g., SQL Server, MySQL Workbench).

Run queries from Global House Purchase Analytics.sql.

Use the KPI dashboard queries as a source for visualization tools (Power BI, Tableau, etc.).

# 📈 Potential Extensions

Build a Power BI Dashboard using the final KPI query.

Perform predictive modeling on purchase probability using Python/R.

Extend dataset with real estate time-series data for trend analysis.
